this is track.
break that 8 track limit in your DAW.
also yes, very creative name, i know
track is a free plugin developed as a workaround to track count limits in DAWs (like Ableton Live Lite)
to get started with using track, simply add the plugin to any track within your host DAW
you can then create tracks to play audio clips, as well as hosting other plugins with delay compensation
